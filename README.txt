== Description ==

MWB Salesforce WooCommerce integration helps the sellers integrate their WooCommerce store with Salesforce in just one click. This integration will help you to upload your products to Salesforce in a single click. For more features checkout the [PRO](https://makewebbetter.com/product/salesforce-woocommerce-integration-pro) version of the plugin.

= Main Features =

* Connect your WooCommerce store with your Salesforce Account.
* Upload store Products to Salesforce.
* Upload details related to Products on Salesforce.
* Sellers can enable logging for each and every API requests to Salesforce.
* The MWB Salesforce WooCommerce Integration plugin is highly optimized and super flexible.

= Premium Plugin Features =

The [Salesforce Woocommerce Integration PRO](https://makewebbetter.com/product/salesforce-woocommerce-integration-pro) plugin offers following features : 

* **Upload** store products to Salesforce.
* **Upload** store products to Salesforce in PriceBook.
* Create **Account** for Customers on Salesforce.
* Link Customers to a single Account on Salesforce.
* Create **Contact** on Salesforce for your store Customers.
* Create **Contract** on Salesforce for your store Customers whenever an order is placed.
* Create new **Lead** on Salesforce whenever an Order is placed.
* Capture you customers as **Opportunity** whenever an Order is placed.
* Create **Order** on Salesforce along with **Order Products** whenever an order is placed.
* Real-time order details syncing.
* Highly optimized and super flexible.

= MINIMUM REQUIREMENTS  =

* WooCommerce 3.2 or greater
* PHP version 5.6 or greater
* MySQL version 5.0 or greater

== Installation ==

= Automatic installation =

In the search field type "MWB Salesforce WooCommerce Integration” and click Search Plugins. Once you’ve found our ‘MWB Salesforce WooCommerce Integration’ you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

= Manual installation =

The manual installation method involves downloading the plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
